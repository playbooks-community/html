# HTML Blog Layout Starter

A static blog structure with an index page listing posts and a post page template with typography, code blocks, and metadata. No site generator required.

Pure HTML and CSS — no JavaScript, no build step.

## Files

- `index.html` — Post listing with 3 sample entries and a sidebar
- `post.html` — Single post template with prose typography and code blocks
- `styles.css` — Typography-focused styles, readable line length, sidebar layout

## Getting Started

Open `index.html` in your browser. Copy `post.html` for each new post and link to it from the index.
