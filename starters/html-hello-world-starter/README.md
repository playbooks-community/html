# HTML Hello World Starter

The most fundamental web starting point. A single HTML file with a proper document structure, CSS reset, custom properties for basic theming, and a centered layout.

No JavaScript. No dependencies. No build step.

## Files

- `index.html` — Document structure with semantic HTML5
- `styles.css` — CSS reset, `:root` custom properties, centered layout

## Getting Started

Open `index.html` in your browser. That's it.
