# HTML Personal Portfolio Starter

A responsive personal portfolio with an about section, project cards, and a contact section. Mobile-responsive layout using CSS Grid and Flexbox.

Pure HTML and CSS — no JavaScript framework, no build step.

## Files

- `index.html` — Header, About, Projects (3 cards), Contact section
- `styles.css` — Responsive grid layout, card styles, badge variants

## Getting Started

Open `index.html` in your browser. Replace the placeholder name, bio, skills, and project info with your own content.
