# HTML CSS Component Library Starter

A single-page HTML reference for reusable CSS components: buttons, form inputs, cards, badges, and alerts. All styled with CSS custom properties for easy theming.

Pure HTML and CSS — no JavaScript, no build step.

## Files

- `index.html` — Organized sections for each component category
- `styles.css` — Component styles built on CSS custom properties

## Components

- **Buttons** — primary, secondary, outline, danger; small/default/large sizes
- **Form inputs** — text, email, textarea, select, checkbox
- **Cards** — basic, with image, with action footer
- **Badges** — blue, green, yellow, red, gray, purple
- **Alerts** — info, success, warning, error

## Getting Started

Open `index.html` in your browser. Copy the HTML for any component you need, and customize the `:root` custom properties in `styles.css` to match your brand.
