# HTML Landing Page Starter

A marketing landing page with a hero section, features grid, call-to-action, and footer — all in plain HTML and CSS. Uses CSS custom properties for theming and CSS Grid for layout.

No JavaScript. No dependencies. No build step.

## Files

- `index.html` — Hero, features grid, CTA section, footer
- `styles.css` — Responsive layout with CSS Grid and custom properties

## Getting Started

Open `index.html` in your browser. Customize the copy, colors, and content in `styles.css` by editing the `:root` custom properties.
